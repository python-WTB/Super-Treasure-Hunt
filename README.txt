You must install python 3 on you system before you can run this program.
On windows, download python 3 from https://www.python.org/downloads/
On linux run command: sudo apt install python3
To run program on windows, double click on it.
To run on linux, open a terminal, go the directory with program, and run command: python3 (program name).py
To run the .exe file, just double click.
To run the linux excutable, change permissions to allow executing file as program, open terminal, go the file's directory, and run command: ./(filename)
